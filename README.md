## 🚀 快速開始

### 環境要求
- Node.js 16+
- npm 或 yarn

### 安裝與運行

1. **安裝依賴**
   ```bash
   npm install
   ```

2. **設置環境變量**
   在 `.env.local` 中設置：
   ```
   GEMINI_API_KEY=your_gemini_api_key
   ```

3. **啟動開發服務器**
   ```bash
   npm run dev
   ```

4. **訪問應用**
   打開瀏覽器訪問：http://localhost:3000
